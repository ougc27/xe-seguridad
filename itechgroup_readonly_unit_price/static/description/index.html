All In One module

Would you like to deal with the unit price? The manager can set the unit price, and not everyone can change the unit price. This one module provides read-only access to a specific seller for whom you won't allow them to edit the unit price in the sale order, invoice, and purchase order.

Features

  • Simple to deal with the unit price.
  • You can read only access to a specific users, everyone can't edit the unit price in the sales order, invoice and purchase order.
  • No special configuration is needed, just install it and use it.
  • This module will save you significant time, it lessens human exertion.
  • This module is compatible with one of them or all modules (sale order, purchase order, invoice)

Configurations

  • Just select the module you need to make the pricing unit read-only
