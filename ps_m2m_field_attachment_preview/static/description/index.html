Many2Many Binary Field Preview Widget

Enhance your Odoo 17 experience with our Many2Many Binary Field Preview Widget.

Note - To use this module, you need to add the many2many_binary_preview widget in your XML field.

Seamlessly preview multiple binary files in Odoo 17 with our Many2Many Binary Field Preview Widget. This widget enhances file management by allowing easy viewing of attachments directly in your form. Improve your workflow with a user-friendly interface that simplifies the handling of various file types.
