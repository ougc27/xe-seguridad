1. Install the ``l10n_mx_sat`` module and configure FIEL credentials in
   Settings > Invoicing > SAT Connection.
2. In the same section, under **Vendor Bill Download**:
   - Set the **Sync from date** to the earliest date you want to download
     vendor bills from. If left empty, it defaults to 30 days ago.
   - Click **Sync Now** to trigger an immediate download, or wait for the
     automatic scheduled action (runs every 6 hours).
3. Monitor progress in Accounting > Configuration > SAT Downloads.
